В рамках тестового задания была произведена верстка элементов странцы по методологии БЭМ.
В разработке применялись такие интсрументы и библиотеки как:
 JADE, STYLUS, NIB
Для слайдера был переработан сторонний скрипт, для меню написан полностью самостоятельно. 

Версия: IE9+ 
Недостатком верстики является лишняя стрелка в select для IE 9, кастомизация на CSS выбрана для эксперимента, для IE лучше использовать JS плагин.
